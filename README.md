## Project Overview

Lena Garment Store is designed to provide a seamless shopping experience for customers looking for high-quality clothing items. It aims to be user-friendly, visually appealing, and efficient for both shoppers and administrators.

## Features

- Browse and search garment collections
- Detailed product descriptions and photos
- Shopping cart functionality
- Secure checkout experience
- User registration and authentication
- Admin dashboard for managing inventory

   ```bash
   npm install
   ```

